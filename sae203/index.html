SAE 203 - Site Web

Juin 2022

Créer un système de votation en ligne grâce à un design UX/UI et un développement et une intégration web frontend et backend.

Apprentissage AC14.01 : Exploiter de manière autonome un environnement de développement efficace et productif

J'ai créé un dépôt GitHub et j'ai hébergé le projet sur mon hébergement personnel, déjà pour être sur de ne jamais perdre le travail et aussi pour qu'il soit accessible librement par les membres de mon groupe, les professeurs et les autres groupes qui avaient besoin d'aide.

Apprentissage AC14.02 : Produire des pages Web fluides incluant un balisage sémantique efficace et des interactions simples

J'ai produit la page de vote, donc la page principale, seulement en HTML et CSS, avec un design reprenant les couleurs du logo MMI Tarbes 2021/2022. Pour les pages de backoffice, je les ai réalisés en HTML, CSS et j'ai utilisé Bootstrap, elles ont donc un design plus générique. Toutes les pages du site sont responsive.

Apprentissage AC14.03 : Générer des pages Web à partir de données structurées

Les pages affichent des données provenant d'une base de données SQL. Que cela soit la question, les choix, les nombres de votes et tout le backoffice.

Apprentissage AC14.04 : Mettre en ligne une application Web en utilisant une solution d’hébergement standard

Le site de vote est hébergé sur les serveurs de l'iut, accessible seulement en étant connecté sur le réseau de l'iut. Je l'ai aussi hébergé avec mon site web personnel mais il n'est pas connecté à la même base de données donc les données n'ont aucun rapport d'une version à l'autre.

Apprentissage AC14.05 : Modéliser les données d’une application Web

J'ai dessiné, modélisé et créé la base de données avec le moins de tables et de champs possibles. J'ai utilisé des 'Foreign Key' pour relier des données à travers plusieurs tables.
